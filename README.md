Starting the project:

    After cloning the repo on macos:
    
    1. cd Travelo
    2. docker-compose up --build
 
    Mobile:
    1. open -a Simulator
        //After the simulator is open
    2. cd Travelo/travelo_mobile
    3. flutter run
    
    Desktop:
    1. cd Travelo/travelo_agency
    2. flutter run -d macos
